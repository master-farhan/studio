About Me

Hi, I'm Farhan Sadik,
a frontend web developer from Bangladesh.

I enjoy creating websites that look good and work well on all devices. I work with HTML, CSS, JavaScript, React.js, Redux and Tailwind CSS to build modern and clean user interfaces.

I'm always learning new things to improve my skills and stay up to date with the latest web technologies. I like solving problems, building cool features, and making websites fast and easy to use.


Right now, I'm focused on growing as a developer and working on projects that help me learn and improve.



My main skills include:

  • HTML & CSS - for creating clean and structured web pages
  • JavaScript - for adding interactivity and dynamic features
  • React.js - for building fast and scalable user interfaces
  • Redux - for managing complex application state effectively
  • Tailwind CSS - for writing efficient and responsive styles quickly
